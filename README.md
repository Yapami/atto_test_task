## Project is only available on linux

## Start:
```
git clone https://github.com/Yapami/atto_test_task.git
```
```
cd atto_test_task
```
```
mkdir build && cd build
```
```
cmake ..
```
```
make
```

## Then run all binaries in order in separate terminals:
```
./server
```
```
./tcp_client
```
```
./udp_client 0
```
```
./udp_client 1
```
